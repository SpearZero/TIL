# 오늘 한 것 
### SpringBoot
- 간단한 REST CRUD API 만들기
    - GET, POST, PUT, PATCH, DELETE
        - GET : 리소스 조회
        - POST : 리소스 생성
        - PUT/PATCH : 리소스 수정(PUT은 전체, PATCH는 부분 교체)
        - DELETE : 리소스 삭제
- 스프링 부트 내장 서버 바꾸기
