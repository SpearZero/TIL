## Codex

- https://www.udemy.com/course/codex-the-practical-guide/
- 위 강의를 보고 실습

명령어
- /를 치면 다양한 명령어들이 나타난다.
- /model 명령어를 통해 모델들을 설정할 수 있다.

Codex 앱
- https://openai.com/ko-KR/index/introducing-the-codex-app/
- 코덱스 앱을 사용하면 여러 프로젝트/스레드를 병렬로 다룰 수 있다.
- CLI보다 손쉽게 사용 가능하다. 참고로 코덱스 앱 내에서 터미널도 열 수 있다.

사용량 보기
- /status 명령어로 사용량/상태를 볼 수 있다.

설정
- 홈 디렉터리 밑에 .codex/config.toml로 전역 설정 가능
- 프로젝트별 .codex/config.toml로 로컬 설정 가능
보안
- 기본 설정: 샌드박스 환경
    - 파일 접근이 제한된다.
    - 일부 작업은 승인이 필요하다.
    - 네트워크 접근이 제한된다
    - 웹 검색은 기본적으로 cached 모드로 제공되며, live 검색이나 비활성화 여부는 설정으로 조정할 수 있다.
- toml 파일을 설정하거나 또는 /permissions 명령어로 권한/승인 정책을 설정할 수 있다.

프로필
- 자주 쓰는 설정 묶음을 이름 붙여 저장해두고, 필요할 때 불러 쓰는 것이 가능하다.
- 프로필은 CLI에서 명시적으로 선택해서 사용할 수 있다.

다양한 제공자
- 기본적으로 OpenAI 모델을 사용하지만, Ollama(로컬)나 OpenAI 호환 provider도 설정해서 사용할 수 있다.

이전 세션에서 시작
- /resume을 이용하면 이전의 작업했던 세션에서 작업이 가능하다

컨텍스트 윈도우
- 대화가 길어졌을 때 /compact 명령어로 이전 대화를 요약해 컨텍스트를 줄일 수 있다.
