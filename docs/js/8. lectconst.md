## let
- function scope
- 호이스팅이 적용된다.

## let, const
- block scope
- 호이스팅이 적용되지 않는다. 초기화 하기전에 변수에 접근하면 ReferenceError 발생
- const는 선언과 동시에 초기화 해야함, 값을 재할당 할 수 없다.
<pre>
    <code>
        let i = 5;
        for(let i = 0; i < 3; i++) {
            console.log(i);
        }
        // 5
        console.log(i);

        var k = 5;
        for(var i = 0; i < 3; i++) {
            console.log(i);
        }
        // 3
        console.log(k);

        //////////////////////////////////////////

        {
            const a = 1;
            let b = 2;
            var c = 3;  // 외부에서 접근 가능
        }

        // 에러 발생
        console.log(a + " : " + b);

        
    </code>
